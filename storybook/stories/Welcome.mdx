# Welcome to the shadcn-native Storybook

This where you can preview and control components all in your browser.

If you are looking for the documentation, you can find it [here](https://shadcn-native.moveinready.casa/docs). This Storybook only includes basic documentation through autodocs.

If you want, you can use the Storybook as a starting point for your design system's storybook. See the [source code](https://github.com/shadcn-ui/shadcn-native/tree/main/storybook) and clone it to get started.

For more information on how to use the Storybook, see the [documentation](https://storybook.js.org/docs/get-started/frameworks/react-native-web-vite?renderer=react-native-web).
