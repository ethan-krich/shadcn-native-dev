# Tooltip

A popup that displays information related to an element when the element receives keyboard focus or the mouse hovers over it.

<Preview component="tooltip" />

_Currently, this component is only available on web. This is because of the lack of a cursor on mobile devices. We are working on a native optimized tooltip component._

See:
